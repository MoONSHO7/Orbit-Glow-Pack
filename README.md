# Orbit Glow Pack

A **media pack** of animated icon glows for [Orbit](../Orbit). It bundles the glow textures and registers them with **LibOrbitGlow-1.0** (embedded in Orbit) — it contains no engine code of its own. Think of it as the glow equivalent of a LibSharedMedia font/texture pack: Orbit-Glow-Pack *provides* media, LibOrbitGlow *plays* it.

## How it works

`Register.lua` runs after Orbit (`## Dependencies: Orbit`), grabs `LibStub("LibOrbitGlow-1.0")`, and calls `lib:RegisterGlow(name, def)` for each bundled glow:

```lua
LCG:RegisterGlow("pinring", {
    layered = true,                                              -- body (BLEND) + core (ADD) two-layer
    path    = "Interface\\AddOns\\Orbit-Glow-Pack\\Textures\\orbit-glow-pinring",
    rows = 6, cols = 5, frames = 30,                             -- 30-frame 5x6 flipbook (Blizzard layout)
    shapes  = { square = true },                                 -- corner shapes available
    source  = "Orbit-Glow-Pack",
})
```

LibOrbitGlow then resolves each phase as `<path>-<phase>-<shape>[-core].tga` (phase = `start|loop|end`, layer = body or `-core`) and any consumer plays it with:

```lua
LibStub("LibOrbitGlow-1.0").Proc:Start(frame, { glow = "pinring", color = {r,g,b,a} })
```

If this pack isn't installed, those glow names simply aren't registered — LibOrbitGlow falls back to its built-in baseline (`blizzard`, the WoW proc atlas), so callers degrade gracefully.

## Textures

`Textures/orbit-glow-<type>-<phase>-square[-core].tga` — 72 glow types × 3 phases × 2 layers = 432 atlases, each a 30-frame `5×6` flipbook at 128px/frame (Blizzard 2× density). Straight-alpha grayscale, so LibOrbitGlow recolours them via `SetVertexColor`. Generated by `.scripts/make-glow-pack.py` / `make-glow-pack2.py` in the Orbit repo (`GLOW_TEXDIR` pointed here).

Currently square-only; squircle/round shapes are a planned addition (the `shapes` table and the `-<shape>` path segment already accommodate them).

## Adding a glow

Generate its `orbit-glow-<name>-{start,loop,end}-square[-core].tga` set into `Textures/`, then add `"<name>"` to the `GLOWS` list in `Register.lua`.
